# genders
A JSON library with genders and translations.

Format:

```js
require('genders')
=>
{
  // english names only here
  "genders": [
    "Agender",
    "Androgyne",
    "Androgynous",
    …
  ],
  "translations": {
    "de": {
      // translation for some of the english names
      "Androgyne": "Androgyn",
      "Androgynous": "Androgyn",
      …
    },
    …
  }
}
```
